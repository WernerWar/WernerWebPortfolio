Game with HTML, CSS and Javascript

This is a Scripting Project where I was tasked to make a playable Game using HTML, CSS, and Javascript.

I decided to make a Space Invaders kind of game on a canvas, but instead of a Spaceship and Invaders, it's a Frog and Flies. I started by making the playable character Class and added an Event listener to move it around, then I made a Grid Class to spawn grids of enemies, after that, I made the playable character and enemies able to shoot, be destroyable, drew the sprites, and a Backround with HTML and CSS.
