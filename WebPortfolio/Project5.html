2D Game with C# in Unity

This is a Game Production Project where I was tasked to make a playable 2D Game with a group member in a Game Jam using C# in Unity.

We decided to make a 2D Tower Defense game in Unity, I was in charge of Programming, Producing, and Game Design, and my group member of the Art/Asset implementation and Sound. I started by making the Grid where the player can build turrets, and then I made enemies that move to Path points through the Grid, afterward, I made different Turrets that can be built and a Shop UI where the player can select the Turret they want to buy, I made an Upgrade system and UI for the turrets that increase their power, finally, I made the Menu, Pause and Game Over UI screens work.
