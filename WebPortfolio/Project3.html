2D Game with C# in Unity

This is a Game Production Project where I was tasked to make a playable 2D Game using C# in Unity.

I decided to make a 2D Platformer game in Unity. I started by making the playable character and its movement and shooting C# scripts, then I made the map layout with Coins and Power Ups to be collected with Collisions and Triggers, afterward, I made the enemy AI C# script and drew the playable character, enemies, and Background sprites, finally, I made the Menu, Pause, and Game Over UI.
