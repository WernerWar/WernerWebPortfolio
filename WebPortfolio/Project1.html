Image with HTML and CSS

This is a Scripting Project where I was tasked to make an animated Image using HTML and CSS.

I decided to make an Anime character that appears out of smoke. I started by making Smoke objects in HTML using div tags and placed them in the Foreground, Middle ground, and Background, I styled these Smoke objects by coloring them gray and shaping them circular as well as animated them to change their position, opacity, scale, and blur.
